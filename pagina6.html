Planilla caso de uso 6: Registrarse

Caso de uso Registrarse
Identificador CU006
Actores Usuario no registrado
Tipo Secundario
Referencias ---
Precondiciones Ninguna
Postcondiciones Iniciar sesión
Descripción El usuario ingresa sus datos para que puedan ser almacenados en la base de dato y por ende el usuario entra al sistema
Resumen ----
Flujo normal
Nro Actor Sistema
1 En la página principal, en la parte
superior derecha, hace clic en “Registrarse”.
2 Se abre una nueva pestaña mostrando las casillas para ingresar los datos:
- Nombre de usuario
- Email
- Repetir email
- Contraseña
- Repetir contraseña
- Número de teléfono
3 Le da clic a “Registrarse”
4 Se realiza la validación de los datos
5 Los datos son guardados en la base de datos
6 Se le muestra página principal, con el chat, ya con su perfil creado
Flujo alternativo
4 Se realiza la validación de datos
4.1 Los datos no son válidos
4.1.1 Se le mostrará un mensaje de error en cada correspondiente indicando que deben ser corregidos
4.1.2 Realiza los cambios
4.1.3 Vuelve al paso 3
